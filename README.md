# Website Pengumpulan Tugas

## Problem
Guru kesulitan mengumpulkan dan mengecek tugas siswa via WhatsApp.

## Solusi
Website sederhana dengan 2 role: Siswa dan Guru.

## Fitur MVP
- Siswa: Kumpul tugas + lihat status
- Guru: Login, lihat semua tugas, download, hapus, cari

## Teknologi
HTML, CSS, JavaScript. Tanpa database dulu biar cepat.

## Cara Jalankan
Buka index.html di browser.

## Yang Saya Pelajari
Belajar tentang localStorage, memisahkan role user, dan berpikir MVP.
